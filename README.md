# ATM Interface
A simple Java-based ATM Interface that allows users to check balance, deposit money, and withdraw money using Object-Oriented Programming (OOP) principles with input validation and a menu-driven interface.

## Features
- Check Account Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Menu-Driven Interface
- Encapsulation using Classes and Objects

## CONCEPTS USED:
- Scanner Class
- OOP Concepts
  - Classes & Objects
  - Encapsulation
  - Constructors
  - Methods

## Project Structure
- `BankAccount` Class
  - Stores account balance
  - Handles deposit and withdrawal operations
  - Provides balance information

- `ATM` Class
  - Displays ATM menu
  - Accepts user input
  - Performs ATM transactions

- `ATMInterface` Class
  - Main class
  - Creates objects and starts the application
  - 
### HOW TO RUN:
### Steps

1. Clone the repository:

git clone <repository-url>
```

2. Navigate to the project directory

3. Compile the Java file:
```bash
javac atm_interface.java
```

4. Run the program:
```bash
java ATMInterface
```

## How It Works
1. The program starts with an initial account balance.
2. The user selects an option from the ATM menu.
3. The selected transaction is processed.
4. Appropriate messages are displayed.
5. The menu continues until the user chooses Exit.
